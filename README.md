## Harvard Extended CS50 course works

## Run

- `pipenv run python -m <module>`
- Some keras (NN) models are saved and retrieved from `models/` folder.

## Model Diagnostics

### Tensorboard

- `pipenv run tensorboard --logdir <path>`
